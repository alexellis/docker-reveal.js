What is docker?

A hot virtualization tool for running applications in an isolated, scriptable and repeatable space.

View from 10,000ft?

  • command-line tool: starts, manages and builds images of Linux applications
  • trendy/disruptive but not new
  • tools are multi-platform
  • Not new - UNIX BSD jails / OpenVZ
  • ever heard of a VPS?

what is a container?

  • Conceptually - refers to shipyard - random sized objects
  • naturally portable - lorry/ship/train
  • An image is built as a series of files/changes/steps executed. A container is an instance of that image during runtime.
